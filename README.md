# 🐻🍺 Bear Goes Pub

A charming, funny 2.5D browser game where you control Bear, a chaotic cat, exploring a cosy British pub and completing ridiculous quests.

## 🎮 Game Features

### Gameplay
- **Exploration-based**: Explore a detailed British pub and surrounding village
- **Quest System**: 6 progressive quests with British humour
- **Interactive NPCs**: 9 unique characters with personality and dialogue
- **Environmental Interactables**: 20+ objects to interact with
- **Easter Eggs**: 10+ hidden jokes and secrets

### Characters

**Bear** - Your chaotic feline protagonist
- Loves snacks and attention
- Confident swagger walk
- Accidentally causes problems

**Supporting Cast:**
- **Toby** - Nervous ginger cat (hidden throughout the pub)
- **Reg the Landlord** - Stern but ridiculous pub owner
- **Old Jim** - 20-year regular in the same chair
- **Weather Wendy** - Obsessed with weather forecasts
- **Footy Frank** - Football fanatic
- **Queue Steve** - Queues for no reason
- **Doris** - "Back in MY day..." nostalgia lover
- **Pigeon Pete** - Feeds pigeons, names them all
- **Vera the Barmaid** - Professional, knows everything
- **Tea Tanya** - Tea solution for everything

### Quests

1. **Find Toby** - Toby ran away from pigeons. Search the pub.
2. **Crisp Delivery** - Deliver packets to demanding NPCs
3. **Pub Cleanup** - Collect: keys, wallet, glasses, remote
4. **Tea Crisis** - Help recover emergency tea supplies
5. **The Mysterious Queue** - Investigate why everyone is queuing
6. **Pub Legend** - Complete all quests for ultimate status

### Locations

**Pub Interior:**
- Sticky wooden tables
- Ridiculous carpet patterns
- Fireplace with weird decor
- Fruit machine
- Dartboard
- Bar area
- Pub toilets
- Beer garden with fence

**Village Area:**
- Bus stop with bench
- Red telephone box
- Corner shop (Greggs + Tesco meals)
- Terraced houses
- Random pigeons
- Grey British sky

### Controls

**Desktop:**
- `WASD` - Move Bear around
- `Mouse` - Look around (camera follows Bear)
- `E` - Interact with nearby objects/NPCs
- `P` - Pause/Resume
- `SPACE` - Jump (reserved for future versions)

**Mobile:**
- **Left side**: Virtual joystick for movement
- **Right side**: Interaction and pause buttons
- **Touch-friendly** UI with large buttons
